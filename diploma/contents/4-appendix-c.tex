\appendixsection{Приложение со списком использованных источников}

В данном приложении приведен перечень основных источников, использованных при
подготовке выпускной квалификационной работы и разработке программного
комплекса.

\begin{enumerate}
    \item Bitcoin Core RPC Documentation.
    \item The Rust Programming Language.
    \item PostgreSQL 16 Documentation.
    \item Docker Documentation.
    \item Docker Compose Documentation.
    \item Prometheus Documentation.
    \item testcontainers for Rust.
    \item Business Process Model and Notation Version 2.0.2.
    \item Unified Modeling Language Version 2.5.1.
    \item The C4 Model for Visualising Software Architecture.
    \item ГОСТ 7.32-2017. Отчет о научно-исследовательской работе. Структура и правила оформления.
    \item ГОСТ Р 57100-2016. Системная и программная инженерия. Описание архитектуры.
    \item ГОСТ Р ИСО/МЭК 25010-2015. Модели качества систем и программных продуктов.
    \item Формы документов, оформляемых при проведении государственной итоговой аттестации МАИ.
\end{enumerate}

Подробное библиографическое описание указанных источников приведено в основном
списке литературы пояснительной записки.
