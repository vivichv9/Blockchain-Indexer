\abstract % Структурный элемент: РЕФЕРАТ

\keywords{ОНЧЕЙН-ДАННЫЕ, ИНДЕКСАЦИЯ ДАННЫХ, БЛОКЧЕЙН BITCOIN, MODULAR MONOLITH, STATELESS-СЕРВИС, RUST, POSTGRESQL, REST API, JOBS, MEMPOOL, REORG, UTXO}

Выпускная квалификационная работа посвящена разработке системы индексации
ончейн данных сети Bitcoin. Актуальность темы определяется необходимостью
выделения специализированного слоя хранения и выдачи данных блокчейна,
который обеспечивает предсказуемую производительность чтения, устойчивость к
изменению канонической цепи и удобный интерфейс взаимодействия для
прикладных систем.

Объектом разработки является программный комплекс
\texttt{Bitcoin Blockchain Indexer}, реализованный в архитектуре модульного
stateless-монолита. Предметом исследования выступают методы индексации
блоков, транзакций, адресных балансов и mempool-данных, а также способы
организации безопасного и управляемого доступа к ним.

Цель работы состоит в проектировании и реализации системы, которая
получает данные из Bitcoin Core по JSON-RPC, нормализует их, сохраняет в
PostgreSQL и предоставляет через REST API, административную панель и
командный интерфейс. В ходе работы были сформулированы требования к системе,
спроектированы архитектура и модель данных, реализованы
jobs индексации, обработка reorg, синхронизация mempool, система метрик и
контур контейнерного развёртывания.

Практический результат работы представляет собой репозиторий с backend на
Rust, миграциями базы данных, документацией по модулям, административной
панелью на React/Vite, Python CLI и набором интеграционных тестов. Полученное
решение может использоваться как базовый инфраструктурный компонент для
аналитических и сервисных систем, работающих с ончейн-данными Bitcoin.
